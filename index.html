Desenvolvimento do projeto

Sprint 1

inicio

O projeto foi inteiramente programado na IDE VScode, desenvolvido usando a linguagem JavaScript na versão mais atualizada - 30/12/2024, junto da linguagem CSS que foi usado para desenvolver o design da pagina

Sprint 2

JavaScript

As tecnicas de programação incluem : a utilização de um "header" para incluir a barra de navegação no topo da pagina, seguido da tag "a" para adicionar as seções que estão na barra de navegação. Foi utilizado o comando "href" para linkar as seções da barra de navegação com o conteudo da pagina. O conteudo dentro destes balões foi formatado utilizando as tags "H2 e H3" para formatação do texto que esta descrito. Para a inclusão das imagens, foi usado o comando "src" com imagens indereçadas e instaladas dentro da pasta do projeto.

Sprint 3

desenvolvimento em css

A parte da programação em CSS, foi utilizado o comando "font-family" junto da fonte "star wars" para criar o design das "h2 e h3" da pagina, incluindo o comando "margin-left" como atributo para criar o espaçamento. Para criar a borda esbranquiçada em volta das caixas foi utilizado o atributo "box-shadow" com "10px" de valor, incluindo também o atributo "border-radius" com "10px" de valor. A fonte dos paragrafos foram formatadas utilizando as fontes "poppins e sans-serif" com "20px" de valor. Cada imagem contem sua formatação unica no script utilizando dos comandos "height","width" e "margin-left".
